Acessibilidade

Dicas de Acessibilidade para Cores

Contraste Mínimo

Certifique-se de que o contraste entre o texto e o fundo atende às diretrizes WCAG 2.1:

  • Texto normal: 4.5:1 (nível AA)
  • Texto grande: 3:1 (nível AA)
  • Texto normal: 7:1 (nível AAA)

Evite Dependência Exclusiva de Cores

Não dependa apenas de cores para transmitir informações. Use ícones, texto ou padrões adicionais para diferenciação. Exemplo:

  • Ícones para indicar sucesso () ou erro ().
  • Texturas ou padrões para distinguir gráficos ou tabelas.

Teste de Daltonismo

Simule como suas cores aparecem para pessoas com diferentes tipos de daltonismo (protanopia, deuteranopia, tritanopia). Ferramentas úteis incluem:

  • Toptal Color Filter
  • Color Oracle

Uso de Cores Seguras

Utilize cores que funcionem bem em telas e dispositivos variados. Evite tons muito claros ou muito escuros para elementos pequenos.

Modos Claro e Escuro

Implemente opções de modo claro e escuro para melhorar a experiência do usuário. Garanta bom contraste em ambos os modos.

Paletas de Cores Consistentes

Mantenha uma paleta de cores consistente em todo o site para facilitar a navegação e reduzir confusão.
